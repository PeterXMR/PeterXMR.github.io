# PeterXMR.github.io

My personal hobby-project portfolio — a single static page served by GitHub Pages,
styled like a terminal (black background, green text, no build step).

🌐 **Live:** https://peterxmr.github.io

## What's here

- `index.html` — the whole site (HTML + inline CSS, no dependencies, no JS)

## Edit / deploy

It's one file. Edit `index.html`, commit, and push to `main` — GitHub Pages redeploys
automatically. To add a project, copy an existing `<article class="project">` block and
fill in the name, tags, description, and links.
